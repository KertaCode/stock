# Stock Boulangérie
Une application de Gestion du Stock dans une Entreprise de Production. Elle gère les processus suivant :
### L'Approvisionnement
Du fournisseur vers les entrepros de l'Entreprise
### L'Utilisation des matières prémières
Mouvements entre la Direction de Fabrication et le Magasin matières Prémières
### La Fabrication
Mouvements entre la Direction de Fabrication (production) et le Magasin Produits Finis
### Les Ventes
Partant de la distribution dans les dépôts relais, les commandes des clients, le paiement des commandes et la livraison
